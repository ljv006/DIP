This program realize the discret Fourier Transformation and smooth or sharpen a picture by using frequency filter. 
You should first put the picture in the folder where you put the cpp file. 
You can enter "dft" to start dft and you will see the source image,its spectrum image and the image after dft.
You can also enter "ff" to start frequency filtering. There are two choices. You can enter "1" to 
smoothing and enter "2" to sharpen the picture.
ps:It will take a few time to finish the task. Please wait patiently.
